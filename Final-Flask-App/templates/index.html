Instructions For Uploading Local Audio Files

  • For each audio upload button, select one local audio file
  • Audio files must be less than 10 seconds
  • Audio files must be in 'm4a' file types
  • Once file are uploaded, click 'Save Files'
  • Once file are saved, click 'Verify Uploaded Speakers'
  • Wait for results
